# 更新日志

这个文件记录项目的重要变更。

版本号在可行范围内遵循语义化版本约定。

## [0.1.0-beta] - 2026-04-10

### 新增

- 补充首个公开 Beta 版本的中英文说明文档
- 增加发布前手工端到端验证清单
- 新增 `scripts/dev-smoke-test.sh`，用于本地快速自测
- 新增 GitHub Actions CI，在 `push` 和 `pull_request` 时自动运行 smoke test

### 调整

- 完善公开 GitHub 仓库的安装说明与初始化引导
- 明确 Beta 状态、私有仓库优先试用建议与已知限制
- 加强同步脚本的安全性，包括更保守的拉取策略、同步仓库有效性检查与敏感字段处理

### 备注

- 在大范围推广前，仍建议先基于真实 GitHub 私有仓库完成一轮端到端验证
- Gitee 兼容性按设计支持，但在正式宣称前仍建议结合真实场景自行回归验证
